Bienvenue dans la documentation d'OC-lettings !
===============================================

Orange County Lettings est un loueur de biens immobiliers. Sur le site,
vous pouvez consulter plusieurs lieux de location ainsi que les profils
des utilisateurs.

Technologies et langages de programmation
-----------------------------------------

Cette application est programmée avec Python et Django.

Les données sont stockées dans la base de données SQLite3.

Les différentes plateformes utilisées sont :

   - GitHub, pour sauvegarder le code et chacune de ses versions,
   - Sentry, pour surveiller les performances du site,
   - Docker, pour une conteneurisation du code,
   - CircleCI, pour une intégration et une livraison continues du code,
   - Render, pour le déploiement de l'application.
